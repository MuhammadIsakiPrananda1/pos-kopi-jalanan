# ☕ KOPI JALANAN GANK - Premium Digital POS

![Logo](assets/images/logo.png)

A high-performance, minimalist, and premium Digital Cashier (Point of Sale) application designed specifically for **KOPI JALANAN GANK**. Built with **Flutter**, it offers a seamless offline-first experience with a focus on speed and modern aesthetics.

---

## ✨ Key Features

- **🚀 Fast Cashier Interface**: Streamlined checkout process with real-time cart management.
- **📊 Advanced Analytics**: Daily, weekly, and monthly sales reports with visual charts.
- **💰 Finance Management**: Integrated income and expense tracking (Manual + Automated Sales).
- **🖨️ Thermal Printing**: Seamless Bluetooth thermal printer support for receipt printing.
- **📦 Inventory System**: Easy-to-use product and category management.
- **🎨 Premium UI/UX**: Dark-themed, glassmorphic design with smooth animations and professional typography.
- **📴 Offline-First**: Reliable SQLite database ensures full functionality without an internet connection.

---

## 🛠️ Built With

- **Framework**: [Flutter](https://flutter.dev)
- **Language**: [Dart](https://dart.dev)
- **Database**: [SQLite](https://sqlite.org) (Local Storage)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **UI Components**: [Google Fonts](https://fonts.google.com), [FL Chart](https://pub.dev/packages/fl_chart)
- **Hardware**: [Print Bluetooth Thermal](https://pub.dev/packages/print_bluetooth_thermal)

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (Latest Version)
- Android Studio / VS Code
- Android Device (API 21+)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/MuhammadIsakiPrananda1/kopi_jalanan_gank.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd kopi_jalanan_gank
   ```

3. **Get dependencies**
   ```bash
   flutter pub get
   ```

4. **Run the app**
   ```bash
   flutter run
   ```
